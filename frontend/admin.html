Tutorial Penggunaan Panel Admin

Workflow Utama (Menambah Pengetahuan AI)

  1. Tambah URL: Buka file `backend/urls_to_scrape.txt` di editor kode Anda. Tambahkan URL artikel atau halaman yang ingin Anda jadikan pengetahuan AI. Setiap URL diletakkan di baris baru.
  2. Scrape URLs: Klik tombol "1. Scrape URLs dari File". Proses ini akan mengambil data teks dan gambar dari semua URL di file tersebut dan menyimpannya ke database.
  3. Tambah Data Manual: Gunakan formulir "Tambah Data Manual" untuk memasukkan teks atau mengunggah file (.pdf, .docx, .pptx) secara langsung sebagai sumber pengetahuan.
  4. Rebuild Index: Setelah scraping atau menambah data manual, klik tombol "2. Rebuild FAISS Index". Ini adalah langkah PENTING untuk membuat data baru bisa dicari oleh AI.

Manajemen Data

Klik tombol "Lihat/Ubah Data Scraped" untuk membuka panel manajemen data. Di sana Anda bisa:

  • Mencari Data: Gunakan search bar untuk memfilter data berdasarkan kata kunci.
  • Melihat Detail: Klik tombol "Detail" untuk melihat seluruh konten dan gambar dari sebuah data.
  • Mengubah Data: Klik "Ubah" untuk mengedit judul atau konten jika ada kesalahan.
  • Menghapus Data: Klik "Hapus" untuk menghapus satu item data dari database.

Pelatihan AI (Memory Bank)

Gunakan bagian "Uji Coba AI" untuk melatih AI secara langsung:

  1. Ajukan pertanyaan untuk melihat bagaimana AI merespons.
  2. Jika jawaban kurang tepat, **klik pada teks jawaban** dan edit langsung di sana.
  3. Setelah jawaban sudah benar, klik tombol "Simpan ke Memori". Jawaban ini akan menjadi prioritas untuk pertanyaan serupa di masa depan.

Tindakan Berbahaya

  • Hapus Index FAISS: Gunakan ini jika Anda merasa pencarian AI error. Anda harus menjalankan "Rebuild Index" lagi setelahnya.
  • Hapus Database (Reset Total): Ini akan menghapus SEMUA data scraping dan memori. Gunakan jika Anda ingin memulai dari awal. Anda harus melakukan scraping dan re-indexing dari nol setelahnya.
